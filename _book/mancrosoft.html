20 Opportunities to Get Certified

Are you a student at The University of Manchester? Would you like to get Microsoft certified on AI, Cloud, Data, Security, CRM, ERP, Azure & 365 this spring / summer? The University are continuing to pilot the introduction of Microsoft Fundamentals Certifications for students.

Figure 20.1: Get hired, stay ahead, be productive and receive the recognition you deserve with Microsoft Certified: Fundamentals. These highly marketable, industry-respected certifications offer training and exams that validate your existing knowledge. Screenshot from learn.microsoft.com. (Nadella 2025) 📈

20.1 Microsoft Fundamentals Certifications

Microsoft Fundamentals Certifications can enhance your CV and LinkedIn profile with signed digital badges. Gaining these certifications typically involves up to 20 hours of study30 and normally costs $99, but they are currently free to University of Manchester students as part of an ongoing pilot project. These highly marketable, industry-respected certifications offer foundational training and exams that provide a practical overview of Microsoft technology. They can help validate and extend your existing knowledge. Choose from the following badged certifications:

  1. Microsoft Azure Cloud Fundamentals: AZ-900
  2. Microsoft Azure AI Fundamentals: AI-900
  3. Microsoft Azure Data Fundamentals: DP-900
  4. Microsoft Security, Compliance and Identity Fundamentals: SC-900
  5. Microsoft Power Platform Fundamentals: PL-900
  6. Microsoft 365 Fundamentals: MS-900
  7. Microsoft Dynamics 365 Fundamentals Customer Relationship Management (CRM): MB-910
  8. Microsoft Dynamics 365 Fundamentals Enterprise Resource Planning (ERP): MB-920

Each certification requires that you:

  • Commit to ~20 hours31 of online study time per certification
  • Attend one in-person 50 minute exam per certification in Manchester in before June, see bit.ly/manchestersoft

There’s more details on these exams, and others like them, in section 23.4.

20.2 Interested?

If you’re interested see the documentation at bit.ly/manchestersoft and email flexiblelearning@manchester.ac.uk
